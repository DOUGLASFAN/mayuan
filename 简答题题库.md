# 马原简答题高频考点完整标准答案（背诵版）
## 一、唯物论
### 1. 物质和意识的辩证关系
1. **物质决定意识**：物质第一性，意识第二性；物质是世界本原，意识是人脑对客观物质的主观映象，物质不以人的意志为转移。
2. **意识对物质具有能动反作用**：
    ① 意识具有目的性、计划性；
    ② 意识具有创造性，能透过现象把握本质；
    ③ 意识指导实践改造客观世界；
    ④ 意识调控人的生理活动。
3. 方法论：一切从实际出发，实事求是；充分发挥主观能动性。

### 2. 客观规律性与主观能动性辩证统一
1. **尊重客观规律是正确发挥主观能动性的前提**。规律客观，不能创造、消灭、改造，违背规律必然失败。
2. **充分发挥主观能动性是认识、利用客观规律的必要条件**。规律隐藏在事物内部，只有主动实践思考才能把握。
3. 二者统一于实践。既要反对唯意志论（夸大主观），又要反对宿命论（否定主观能动）。

### 3. 意识的能动作用（意识对物质的反作用）
1. 意识活动具有目的性和计划性。人行动前预设目标、方案。
2. 意识活动具有创造性。能加工改造感性材料，把握事物本质、预见未来。
3. 意识具有指导实践改造客观世界的作用，这是意识能动作用最突出体现。
4. 意识具有调控人的生理、心理活动的作用。

### 4. 运动与静止辩证关系
1. **运动是绝对的**：运动是物质根本属性、存在方式，一切事物永恒变化。
2. **静止是相对的**：静止是运动特殊状态，包括空间位置不变、根本性质暂时不变。
3. 动中有静，静中有动，任何事物都是绝对运动与相对静止统一。
4. 反对形而上学不变论、相对主义诡辩论。

## 二、辩证法（重中之重）
### 1. 为什么对立统一规律是唯物辩证法的实质和核心
1. 对立统一规律揭示了事物普遍联系的**根本内容**和事物发展的**内在动力**。
2. 它是贯穿唯物辩证法其他规律、范畴的**中心线索**，质量互变、否定之否定都是矛盾运动表现。
3. **矛盾分析法**是我们认识世界、改造世界的根本认识方法。
4. 是否承认事物内部矛盾，是辩证法和形而上学**根本分歧**。

### 2. 矛盾普遍性与特殊性辩证关系
1. **矛盾普遍性**：矛盾无处不在、无时不有。
2. **矛盾特殊性**：不同事物、不同阶段、矛盾各方面各有特点。
3. 二者相互联结：普遍性寓于特殊性之中，通过特殊性表现；特殊性包含普遍性。
4. 二者在一定条件下相互转化。
5. 方法论：坚持具体问题具体分析；是马克思主义基本原理同中国实际结合哲学依据。

### 3. 矛盾同一性和斗争性辩证关系
1. **同一性**：矛盾双方相互依存、相互贯通，一定条件下相互转化。
2. **斗争性**：矛盾双方相互排斥、相互分离。
3. 同一性是相对的、有条件的；斗争性是绝对的、无条件的。
4. 同一性为斗争提供场所，斗争推动事物变化；矛盾双方又同一又斗争推动事物发展。

### 4. 辩证否定观（扬弃）
1. 否定是事物**自我否定**，内部矛盾运动结果。
2. 否定是事物发展的环节，旧事物灭亡、新事物产生必经之路。
3. 否定是事物联系的环节，新事物保留旧事物合理因素。
4. 实质是**扬弃**：既克服消极过时内容，又保留积极合理因素。
5. 反对全盘肯定、全盘否定的形而上学否定观。

### 5. 联系的特点
1. **客观性**：联系是事物本身固有，不以人的意志转移。
2. **普遍性**：事物内部、事物之间、整个世界普遍联系，孤立事物不存在。
3. **多样性**：因果、整体部分、偶然必然、本质非本质等多种联系。
4. **条件性**：任何联系依赖一定条件，条件可变。

### 6. 发展的实质
发展的实质是**新事物产生，旧事物灭亡**。
新事物符合客观规律、具有远大前途；旧事物违背规律、日趋灭亡。发展是前进上升运动，不等同于一切变化。

## 三、认识论
### 1. 实践对认识的四个决定作用
1. **实践是认识的来源**：一切认识都来自实践，直接经验、间接经验归根到底源于实践。
2. **实践是认识发展的动力**：实践提出新课题、提供工具、锻炼人的认识能力。
3. **实践是检验认识真理性的唯一标准**。
4. **实践是认识的目的**：认识世界最终是为了改造世界。

### 2. 感性认识与理性认识辩证关系
1. **感性认识是认识初级阶段**：感觉、知觉、表象，反映事物外部现象；
   **理性认识是高级阶段**：概念、判断、推理，反映事物内在本质。
2. 理性认识依赖感性认识，感性是理性基础。
3. 感性认识有待上升、发展为理性认识。
4. 二者相互渗透：感性中有理性，理性中有感性。
5. 统一基础是实践，反对唯理论、经验论。

### 3. 真理的绝对性和相对性辩证关系
1. **真理绝对性**：真理内容客观；人类认识能正确认识无限发展的物质世界。
2. **真理相对性**：认识广度有限，有待扩展；深度有限，有待深化。
3. 二者相互依存，任何真理都是绝对与相对统一。
4. 相对真理不断积累走向绝对真理；绝对真理寓于相对真理之中。
5. 反对绝对主义、相对主义。

### 4. 实践标准的确定性与不确定性
1. **确定性（绝对性）**：实践是检验真理唯一标准；凡经实践检验的真理具有不可推翻性。
2. **不确定性（相对性）**：社会实践受历史条件制约，当下实践不能完全证实/证伪一切认识；实践检验是长期反复过程。
3. 二者统一，不能割裂。

## 四、唯物史观
### 1. 社会存在与社会意识辩证关系
1. **社会存在决定社会意识**：社会存在是社会生活物质条件总和，社会意识是社会存在主观反映；社会存在变化，社会意识迟早变化。
2. **社会意识具有相对独立性**：
    ① 与社会存在发展不完全同步、不平衡；
    ② 内部各形式相互影响；
    ③ 具有历史继承性；
    ④ **对社会存在具有能动反作用（最突出）**：先进意识促进社会发展，落后意识阻碍发展。

### 2. 人民群众是历史的创造者
1. 人民群众是**社会物质财富创造者**，是生产力首要要素，提供人类生存物质基础。
2. 人民群众是**社会精神财富创造者**，一切精神文化素材、实践源泉来自群众。
3. 人民群众是**社会变革的决定力量**，社会革命、改革依靠群众实践推动。
4. 方法论：坚持群众观点、群众路线；发展依靠人民、为了人民、成果人民共享。

### 3. 科学技术在社会发展中的双重作用
#### 积极作用（革命力量）
1. 科技是第一生产力，变革生产工具、劳动者素质、劳动对象，大幅提升生产力。
2. 推动生产关系、社会制度变革，推动社会形态进步。
3. 提升人类认识能力，解放思想，丰富精神文化生活。
#### 消极负面效应
1. 不合理运用造成生态破坏、环境污染、资源枯竭。
2. 引发伦理难题：人工智能、基因技术、数据隐私等风险。
3. 资本主义制度下，科技异化为剥削劳动者、霸权扩张的工具。
4. 自动化、智能化带来结构性失业，拉大贫富差距。
总结：科技作用受社会制度、人的价值观念制约，要合理规制、以人为本发展科技。

### 4. 生产力与生产关系矛盾运动规律
1. **生产力决定生产关系**：生产力状况决定生产关系性质；生产力发展推动生产关系变革。
2. **生产关系反作用于生产力**：适合生产力则促进；不适合则阻碍。
3. 二者矛盾运动：基本适合→不适合→新的基本适合，循环推动社会发展。

## 五、政治经济学（高频简答）
### 1. 价值规律内容、表现形式、双重作用
1. **基本内容**：商品价值量由生产商品社会必要劳动时间决定；商品交换以价值量为基础等价交换。
2. **表现形式**：价格围绕价值上下波动。
3. **积极作用**
    ① 自发调节生产资料、劳动力在各部门分配，配置社会资源；
    ② 刺激生产者改进技术、提高劳动生产率，推动生产力发展；
    ③ 市场竞争优胜劣汰，优化市场主体。
4. **消极局限性**
    ① 调节具有自发性、滞后性，易造成资源浪费；
    ② 容易催生垄断，阻碍新技术推广；
    ③ 引起商品生产者两极分化，加剧贫富差距。

### 2. 商品二因素（使用价值、价值）对立统一
1. **使用价值**：商品能够满足人需要的属性，自然属性，构成社会财富物质内容。
2. **价值**：凝结在商品中无差别人类劳动，社会属性，是商品特有属性。
3. **统一性**：商品必须同时具备二因素，缺一不可。
4. **对立性**：生产者只能获得价值，消费者只能获得使用价值，二者分离才能实现。

### 3. 劳动二重性（具体劳动、抽象劳动）
1. **具体劳动**：特定形式有用劳动，创造使用价值。
2. **抽象劳动**：撇开具体形式无差别人类劳动，形成商品价值。
3. 二者是**同一劳动过程两个方面**，不是两次劳动；劳动二重性是理解政治经济学枢纽。

### 4. 劳动力商品价值与使用价值特殊性
1. **劳动力商品价值构成**
    ① 维持劳动者自身生存的生活资料价值；
    ② 维持劳动者家属生存的生活资料价值；
    ③ 劳动者教育、培训费用。
2. **使用价值最大特殊性**：劳动力使用过程（劳动）能够创造出**比自身价值更大的价值（剩余价值）**，这是货币转化为资本关键。

### 5. 不变资本、可变资本划分依据及意义
1. **划分依据**：资本不同部分在剩余价值生产中发挥的作用不同。
2. **不变资本c**：生产资料，只转移原有价值，不产生剩余价值。
3. **可变资本v**：购买劳动力资本，能创造剩余价值。
4. **意义**：揭示剩余价值真正来源是雇佣工人剩余劳动；为计算剩余价值率提供依据。

### 6. 资本积累的本质与社会后果
1. **本质**：资本家无偿占有工人创造的剩余价值，将其转化为资本，进一步扩大剥削规模。
2. **后果**：
    ① 加剧社会两极分化；
    ② 资本有机构成不断提高，造成相对过剩人口（失业）；
    ③ 激化资本主义基本矛盾。

### 7. 私有制商品经济基本矛盾：私人劳动和社会劳动矛盾
1. 私人劳动：生产资料私有，生产者自主安排生产。
2. 社会劳动：商品用于社会交换，劳动具有社会性。
3. 矛盾表现：劳动产品无法顺利交换时，私人劳动不能转化为社会劳动。
4. 地位：是商品经济一切矛盾根源，决定商品其他矛盾（二因素、劳动二重性等）。

### 8. 资本主义经济危机实质与根源
1. **实质**：生产相对过剩，不是产品绝对太多，而是劳动者购买力不足。
2. **根本根源**：资本主义基本矛盾——生产社会化与生产资料资本主义私人占有之间的矛盾。

## 六、科学社会主义简答
### 1. 社会主义从空想变为科学的两大理论基石
唯物史观、剩余价值学说。
唯物史观揭示社会发展客观规律；剩余价值学说揭露资本主义剥削秘密，二者使社会主义具备科学理论支撑。

### 2. 国家垄断资本主义本质
资产阶级国家政权与私人垄断资本相结合的垄断资本主义，本质是为垄断资产阶级整体利益服务，维护资本主义制度，没有改变资本主义根本性质。
