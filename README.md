# 《马原刷题系统》README.md
## 一、项目介绍
马原刷题网站是一款轻量化线上马克思主义原理刷题工具，专为马原备考人群设计，无需复杂部署，打开网页即可刷题、查看题库、复盘错题，适配期末复习、考研政治基础巩固场景。
项目采用纯前端 + 简易JS服务搭建，部署于GitHub Pages，可直接在线访问使用，支持本地离线运行。

在线预览地址：https://mayuan-pink.vercel.app

## 二、功能特点
1. **多题型完整题库**
    - 简答题库、材料分析题库、马克思主义原理复习题全覆盖；
    - 配套PDF题库文档，可下载打印线下背诵。
2. **在线刷题模块**
    - `index.html`：首页入口，功能导航；
    - `exam.html`：正式答题页面，随机出题作答；
    - `review.html`：错题/历史题目复盘，回看答案解析；
3. **本地一键启动**
    内置`start.bat`批处理文件，Windows双击即可快速启动本地服务，离线刷题无网络限制；
4. **轻量化易部署**
    仅HTML/JS/少量批处理，无后端依赖，支持GitHub Pages、Vercel一键托管；
5. **简洁交互界面**
    低冗余设计，专注刷题场景，加载速度快，手机、电脑端均可适配。

## 四、本地运行教程
### 方式1：一键启动（推荐Windows用户）
1. 克隆/下载本仓库到本地；
2. 进入项目根目录 `D:\AIcode\MayuanExamWeb`；
3. 双击 `start.bat` 文件自动启动本地服务；
4. 打开浏览器访问控制台输出的本地地址即可离线刷题。

### 方式2：手动启动Node服务
CMD中依次执行：
```bash
# 进入项目目录
cd D:\AIcode\MayuanExamWeb
# 启动本地服务
node server.js
```

### 方式3：直接打开静态页面
无需服务，双击 `index.html` 可直接打开基础刷题页面（部分交互功能受限）。
